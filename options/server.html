Server¶

OpenNMT-py REST Server

usage: server.py [-h] [--ip IP] [--port PORT] [--url_root URL_ROOT] [--debug]
                 [--config CONFIG]

Named Arguments¶

--ip

Default: “0.0.0.0”

--port

Default: 5000

--url_root

Default: “/translator”

--debug, -d

Default: False

--config, -c

Default: “./available_models/conf.json”
