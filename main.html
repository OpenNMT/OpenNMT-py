Overview¶

This portal provides a detailed documentation of the OpenNMT toolkit. It describes how to use the PyTorch project and how it works.

Installation¶

Install from pip: Install OpenNMT-py from pip:

pip install OpenNMT-py

or from the sources:

git clone https://github.com/OpenNMT/OpenNMT-py.git
cd OpenNMT-py
python setup.py install

(Optionnal) some advanced features (e.g. working audio, image or pretrained models) requires extra packages, you can install it with:

pip install -r requirements.opt.txt

And you are ready to go! Take a look at the quickstart to familiarize yourself with the main training workflow.

Alternatively you can use Docker to install with nvidia-docker. The main Dockerfile is included in the root directory.
